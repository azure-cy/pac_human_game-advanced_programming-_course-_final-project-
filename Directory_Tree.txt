Pacman_Game/
├── assets/                 # [资源库] 存放图片、字体、音效
├── src/                    # [核心代码库]
│   ├── __init__.py         # 使 src 成为一个 Python 包
│   ├── settings.py         # [配置中心] 常量、颜色、屏幕设置
│   ├── maps.py             # [数据中心] 关卡地图数据 (Grid)
│   ├── sprites.py          # [实体工厂] 玩家、鬼、墙、豆子的类定义
│   ├── level.py            # [关卡逻辑] 地图生成、碰撞处理、胜利失败判定
│   ├── ui.py               # [界面管理] 主菜单、分数显示、结束画面
│   └── game.py             # [主控中心] 状态机，管理 Menu/Level 切换
└── main.py                 # [启动入口] 程序的唯一入口
